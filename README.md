# Python-Basic-15-Problems

*Level 1: Basics & Input/Output*
*1. Hello, Name*  
Ask for the user’s name and print `Hello, <name>! Welcome to Python.`

*2. Even or Odd*  
Take a number as input and print whether it’s even or odd.

*3. Sum of Two Numbers*  
Take 2 numbers as input and print their sum, difference, product, and division.

*4. Celsius to Fahrenheit*  
Convert a temperature from Celsius to Fahrenheit using the formula: `F = C * 9/5 + 32`

*5. Largest of 3 Numbers*  
Take 3 numbers and print the largest without using `max()`.

*Level 2: Loops & Conditionals*
*6. Multiplication Table*  
Print the multiplication table of a number from 1 to 10.

*7. Sum of First N Numbers*  
Take `n` as input and print the sum of 1+2+3+...+n using a loop.

*8. Reverse a Number*  
Take a number like 1234 and print it reversed as 4321.

*9. Prime Number Check*  
Check if a given number is prime.

*10. Count Vowels & Consonants*  
Take a string and count how many vowels and consonants it has.

*Level 3: Lists, Strings & Functions*
*11. Find Max in List*  
Create a list of 5 numbers and find the largest without using `max()`.

*12. Remove Duplicates*  NC
Take a list with duplicates like `[1,2,2,3,4,4]` and return `[1,2,3,4]`.

*13. Palindrome Check*  
Check if a word or phrase reads the same backwards. Ignore spaces and case.  
Example: `racecar` → True, `Hello` → False

*14. Factorial Using Function*  
Write a function `factorial(n)` that returns n!. Test it with `factorial(5)`.

*15. Simple To-Do List*  
Make a mini program with a menu:  
1. Add task  
2. View tasks  
3. Exit  
Store tasks in a list and loop until the user exits.
